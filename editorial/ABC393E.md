## 解き方を思いつくまでの流れ
1. $x$ で割り切れる $A$ の要素が $K$ 個以上存在するもの $\cdots (1)$のうち、 $x$ が最大のものが答え
   - ある $x$ で割り切れる $A$ の要素が何個存在するかを高速で計算する手法が欲しい
   - 約数列挙は $O(\sum_{k=1}^{N} \sqrt N) = O(N \sqrt N) $ かかるので間に合わない
2. 調和級数で $O(N \log N)$ で計算できるものの形を思い出す
   ```cpp
    for(int d = 1; d <= N; d++) {
        for(int n = d; n <= N; n += d) {
            // 何かしらの処理
        }
    }
   ```
3. 最終的に欲しいもの`u[n]`: `A[i] = n`の時の答え
   - ある`d`が $(1)$ を満たしているとき、`A[i] = kd` で表されるものは `d` が答えの候補となる。
   - `t[d]`: `d` で割り切れる $A$ の要素の個数
   - `M`: $A$ の最大の要素
   ```cpp
    for(int d = 1; d <= M; d++) {
        if(t[d] < K) continue;
        for(int n = d; n <= M; n += d) {
            chmax(u[n], d);
        }
    }
   ```  
4. `t[d]` を求める
   - `t[d] = (A に含まれる d の倍数の個数)`とも言い換えられる
   - `s[n]`: $A$ に含まれる `n` の個数
   ```cpp
    for(int d = 1; d <= M; d++) {
        for(int n = d; n <= M; n += d) {
            t[d] += s[n];
        }
    }
   ```
