## ワーシャルフロイド法
計算量：$O(V^3)$
```cpp
template <typename T = int>
void warshall_floyd(vector<vector<T>> &G) {
  int N = ssize(G);
  for (int k = 0; k < N; k++) {
    for (int i = 0; i < N; i++) {
      for (int j = 0; j < N; j++) {
        G[i][j] = min(G[i][j], G[i][k] + G[k][j]);
      }
    }
  }
}
```
