# 直線の作り方

## 基本

直線は $(a,b,c) = (x\text{の係数}, y\text{の係数}, \text{切片})$ で管理する。
また、表記ゆれが生じないように $a \ge 0$ かつ $a, b, c$ が互いに素にしておく。

### ある直線に平行

$k = (a,b)$ に平行で $c = (x_0, y_0)$ を通る直線

$$
\begin{aligned}
    a(y - y_0) &= b(x - x_0) \Leftrightarrow
    bx - ay - bx_0 + ay_0 = 0
\end{aligned}
$$

### ある直線に垂直

$k = (a,b)$ に垂直で $c = (x_0, y_0)$ を通る直線

$$
\begin{aligned}
    a(x - x_0) + b(y - y_0) = 0 \Leftrightarrow
    ax + by - ax_0 - by_0 = 0
\end{aligned}
$$
