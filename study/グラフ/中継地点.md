# 中継地点

## 頂点追加

### 可変テレポーターの例

- 中継地点を通らない
    - 経路: `町1 → ... → 町N`
    - 最短時間: `dist_1[N]`
- 中継地点を一回通る
    - 経路: `町1 → ... → (ある未定端点v) → 町i → ... → 町N`
    - 最短時間: `dist_1[0] + dist_2[i]`
- 中継地点を一回通る
    - 経路: `町1 → ... → 町i → (ある未定端点v) → ... → 町N`
    - 最短時間: `dist_1[i] + dist_2[0]` 
- 未定テレポーターを2回使う
    - 経路: `町1 → ... → (未定端点v1) → 町i → (未定端点v2) → ... → 町N`
    - 最短時間: `dist_1[0] + dist_2[0]`
