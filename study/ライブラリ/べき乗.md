## 高速な整数pow
```cpp
ull powll(ull x, ull n) {
  ull ret = 1;
  while (n > 0) {
    if (n & 1) ret *= x;
    x *= x;
    n >>= 1;
  }
  return ret;
}
```

## 安全なpow
整数範囲でべき乗を求める場合、高々60回程度のループで計算できるのでdouble型でオーバフローしないことを確認してから計算するという処理を入れる

```cpp
ll safe_pow(ll a, ll b) {
  ll res = 1;
  for (ll i = 0; i < b; i++) {
    double dres = res;
    dres *= a;
    if (dres > 2e18) {
      return 2e18;
    }
    res *= a;
  }
  return res;
}
```
